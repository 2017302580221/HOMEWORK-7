### p6

![p6](img/p6.png)

| 目的地址范围      | 链路接口 |
| ----------------- | -------- |
| 00000000~00111111 | 0        |
| 01000000~01011111 | 1        |
| 01100000~01111111 | 2        |
| 10000000~10111111 | 2        |
| 11000000~11111111 | 3        |

```
接口0的地址数量：2^6 = 64个
接口1的地址数量：2^5 = 32个
接口2的地址数量：2^5+2^6 = 96个
接口3的地址数量：2^6 = 64个
```
