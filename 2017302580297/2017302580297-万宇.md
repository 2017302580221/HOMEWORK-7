![P5](P5(2).png)

a. 使用最长前缀匹配的转发表如下：

| 前缀匹配          | 链路接口 |
| ----------------- | -------- |
| 11100000 00       | 0        |
| 11100000 01000000 | 1        |
| 1110000           | 2        |
| 11100001 1        | 3        |
| 其他              | 3        |

b. 链路接口分别为3、2、3，依次通过其他、1110000、11100001 1  前缀匹配得到。
